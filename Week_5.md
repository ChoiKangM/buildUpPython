* 객체 템플릿 : 클래스(`Class`) - 붕어빵 틀
* 실제 구현체 : 인스턴스(`Instance`) - 붕어빵

# 클래스 생성
```python
class Car:
  color = ""
  speed = 0

  def up_speed(self, value):
    self.speed += value

  def down_speed(self, value):
    self.speed -= value
```
